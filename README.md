# Simple Data Analyzer

A Python program that analyzes numerical data from a text file.

## Usage

1. Create a text file with one number per line (e.g., `data.txt`):
   ```
   1.5
   2.3
   4.7
   3.2
   ```

2. Run the program:
   ```
   python alpha.py data.txt
   ```

## Features

- Computes count, sum, mean, min, max, and median
- Handles invalid data gracefully
- Works with decimal numbers

## Requirements

- Python 3.x

## Troubleshooting

- Ensure the file exists and is readable
- Check that numbers are separated by newlines
- Invalid lines are skipped with a warning
